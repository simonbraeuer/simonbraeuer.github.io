# Simon Bräuer - Software Developer Portfolio

A responsive, multi-language software developer portfolio built with Angular 21.

## Features

- **Angular 21** with standalone components
- **Responsive Design** — Mobile-first architecture
- **Multi-Language Support** — German, English, Spanish, Afrikaans, Latin, Klingon
- **SCSS Styling** — Clean, maintainable stylesheets
- **Resume Download** — Direct PDF download
- **GitHub Pages Ready** — Pre-configured deployment

## Quick Start

```bash
# Install dependencies
npm install

# Run dev server
npm start
```

Open `http://localhost:4200` in your browser.

## Build & Deploy

```bash
# Production build
npm run build:prod

# Deploy to GitHub Pages
npm run deploy
```

The site will be available at `https://simonbraeuer.github.io`

## Customization

Edit `src/app/app.component.ts` to modify:
- Hero section text and translations
- Skills and portfolio items
- Contact information
- Language strings

Update colors in `src/app/app.component.scss` (lines 3-6).

## Technology Stack

- **Framework:** Angular 21
- **Language:** TypeScript 5.9.3
- **Styles:** SCSS
- **Build Tool:** Angular CLI
- **Deployment:** GitHub Pages
